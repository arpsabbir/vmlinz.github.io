Install hexojs

Requirements

  • git
  • nodejs

Install from npm

npm install -g hexo-cli

Basic setup

Once you have hexo installed, you can use following commands to initialize the target folder

hexo init BLOG_DIR
cd BLOG_DIR
npm install

Customization

  • configuration
  • scaffolds
  • themes

Writing

new post
hexo new [layout] TITLE

publish a draft
hexo publish [layout] TITLE
